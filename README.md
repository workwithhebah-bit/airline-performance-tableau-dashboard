# Airline Performance Analytics Dashboard

## Project Overview
This project analyzes airline performance data to understand flight trends, airport activity, and operational patterns using Tableau.

## Tools Used
- Tableau Desktop
- Data Visualization
- Data Cleaning

## Dashboard Features
- Arrival and departure analysis
- Airport-wise performance
- Interactive filters
- KPI indicators
